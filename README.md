# news-electron

# Electron桌面端新闻，使用的是网易云新闻API接口获取的新闻数据。

### 安装方式
```
//  使用git 克隆
npm install 

//　安装好依赖之后直接运行
npm run dev

//  打包
npm run build

//  测试
npm test
